## 3. Auditor Interview Prep

Common questions + suggested answer structure:

### Q1: "Walk me through a sample request lifecycle."

A: "User submits via authenticated UI [show flow]. Input goes through PII mask [show code]. Request to Claude via Bedrock VPC endpoint [show diagram]. Response filtered [show filter]. Logged to Langfuse [show trace]. User sees response with citations [show UI]."

### Q2: "How do you ensure model outputs are accurate?"

A: "We measure accuracy quarterly via [eval set link]. CI runs eval on every PR [show workflow]. Production sample 5% scored by LLM judge [show dashboard]. User feedback aggregated [show metric]. Quarterly review by Ethics Committee."

### Q3: "What happens if a user asks for deletion?"

A: "/api/user/delete endpoint triggers deletion across 7 surfaces [list]. Tested end-to-end [show test]. Audit log proves completion [show log]. SLA: 30 days to confirm. Backups: 90-day cycle, documented in DPA."

### Q4: "How do you handle bias?"

A: "Pre-deploy: fairness eval across protected attributes [show metric]. Production: monitoring per group [show dashboard]. If parity diff > 5%, alert + investigation. Quarterly bias audit by external reviewer."

### Q5: "How do you select Claude models?"

A: "Decision matrix in ADR-004 [show]. Haiku for routing/classification. Sonnet for QA. Opus reserved for complex reasoning. Routing layer chooses per query [show code]."

## 5. Continuous Compliance

```python
# Automated compliance checks in CI
def check_compliance():
    issues = []
    
    # Documentation freshness
    for doc in glob("compliance/**/*.md"):
        age_days = (now() - file_mtime(doc)).days
        if age_days > 180:
            issues.append(f"{doc} stale ({age_days}d)")
    
    # Required docs present
    REQUIRED = [
        "compliance/01-governance/ai-policy.md",
        "compliance/02-risk/risk-register.md",
        # ...
    ]
    for r in REQUIRED:
        if not os.path.exists(r):
            issues.append(f"Missing: {r}")
    
    # Risk register has open high-priority items?
    risks = parse_risk_register()
    open_high = [r for r in risks if r.score >= 16 and r.status == "open"]
    if open_high:
        issues.append(f"{len(open_high)} unmitigated high risks")
    
    return issues
```

# .github/workflows/compliance.yml

      - name: Notify if issues
