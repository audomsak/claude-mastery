## 📖 Course Overview

### Month 1: Foundation (Day 1-30)
- Week 1: Claude basics
- Week 2: Applied Claude (writing, data, code, API, Cowork)
- Week 3: Developer tools (Claude Code, MCP)
- Week 4: Agents + first capstone

### Month 2: Enterprise AI Engineering (Day 31-60)
- Week 5: RAG fundamentals
- Week 6: Advanced RAG (hybrid, GraphRAG, agentic)
- Week 7: Frameworks ecosystem (LangChain, LangGraph, LlamaIndex, DSPy)
- Week 8: Claude on cloud (Bedrock, Vertex, Foundry)

### Month 3: Production + Capstone (Day 61-90)
- Week 9: Advanced agents (Computer Use, E2B, Skills)
- Week 10: Multi-agent (CrewAI, LangGraph, AutoGen, A2A intro, memory)
- Week 11: LLMOps (observability, eval, CI/CD, red team, guardrails, cost, runbook)
- Week 12: Enterprise capstone v2 (full deploy)

### Month 4: Specialization & Mastery (Day 91-120)
- Week 13: Voice (LiveKit, ADK) + Document AI (LandingAI)
- Week 14: Compliance (NIST, EU AI Act, PDPA, HIPAA, Green AI)
- Week 15: Vertical playbooks (support, code, legal, finance, health, edu)
- Week 16: Advanced MCP + A2A + Enterprise MCP server build

## 🛠️ Tech Stack Covered

- **LLM**: Claude Opus 4.7 / Sonnet 4.6 / Haiku 4.5
- **Frameworks**: LangChain, LangGraph, LlamaIndex, DSPy, Pydantic AI, CrewAI, AutoGen
- **Vector DBs**: Qdrant, Pinecone, Weaviate, pgvector, Neo4j (graph)
- **Cloud**: AWS Bedrock, Google Vertex AI, Microsoft Foundry
- **Voice**: LiveKit, Google ADK, Deepgram, ElevenLabs
- **Doc AI**: LandingAI Agentic Doc, Claude vision
- **Observability**: Langfuse, Phoenix, OpenTelemetry, Prometheus, Grafana
- **Evaluation**: Ragas, DeepEval, LLM-as-judge
- **Security**: Giskard, PyRIT, Guardrails AI, NeMo Guardrails, Bedrock Guardrails
- **Memory**: Letta, LangMem, Mem0
- **Protocols**: MCP (Model Context Protocol), A2A (Agent-to-Agent)
- **Infra**: Terraform, ECS Fargate, RDS, VPC endpoints

## 📋 Prerequisites

- Programming experience (Python preferred)
- Familiarity with REST APIs, basic cloud concepts
- For Month 3+: backend dev + some DevOps
- For Month 4: enterprise architecture exposure helps
